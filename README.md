# chessboard-app

This is a small online chessboard app I made (to play checkers).

---

# Features:

- **2** Different Pieces
- **1** Board
- **0** Rule Checking
- Some **online** functionality

---

To run:

```bash
bun run index.js
```

This project was created using `bun init` in bun v1.3.9. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
